# **App Name**: ClipBook-Online

## Core Features:

- Note Editor: Clean, intuitive interface for creating, editing, and organizing notes.
- Note Organization: Categorize notes using tags and custom labels.
- Offline Access: Offline mode support for accessing notes without an internet connection using service workers.
- PWA Support: Progressive Web App (PWA) installable on desktop and mobile devices.
- AI Summarization: Summarize long notes into key bullet points using AI.
- Tag Suggestion: Intelligently suggest relevant tags based on note content, using a tool to decide which suggestions are relevant.
- Customization: Ability to customize the theme (light/dark mode, font size, etc.).

## Style Guidelines:

- Primary color: Forest green (#347C58) to evoke a sense of calm and focus.
- Background color: Light beige (#F5F5DC), offering a soft and non-distracting canvas.
- Accent color: Warm gold (#D4AF37), for highlights and important actions.
- Font: 'Literata', a transitional serif font for comfortable reading and a literary feel.
- Use simple, line-based icons for clarity and easy recognition.
- Clean, minimalist design with ample whitespace to reduce visual clutter.
- Subtle animations and transitions to enhance user experience (e.g., smooth scrolling, fade-in effects).
